===============================
    Password Manager - v1.0
===============================

A modern, lightweight Password Manager built with Python and Tkinter for securely storing and managing your passwords offline.

-------------------------------
FEATURES
-------------------------------
- Save website, email/username, and password entries
- Generate strong passwords automatically
- Copy password to clipboard
- View, search, and delete saved passwords
- Show/Hide password toggle
- Clean, tabbed interface (Add / View)
- Light, Blue, and Dark theme support

-------------------------------
INSTALLATION
-------------------------------
1. Unzip the folder.
2. Run the file: PasswordManager.exe

No installation required. Just run and use!

-------------------------------
SYSTEM REQUIREMENTS
-------------------------------
- Windows 10 or higher (64-bit)
- Python 3.8+ (if running from source)
- 100 MB disk space

-------------------------------
USAGE INSTRUCTIONS
-------------------------------
- Double-click `PasswordManager.exe` to launch the app
  (or run `main.py` if using source)
- Fill in website, email, and password
- Click "Generate Password" to auto-fill a strong one
- Click "Save Password" to store it locally
- Switch to the "View Saved" tab to see entries
- Use the search bar to quickly find a password
- Select an entry and click "Delete" to remove it
- Use the theme toggle for Dark / Blue / Light mode
